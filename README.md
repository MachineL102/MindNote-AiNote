# MindNote

MindNote 是一款本地优先的画布式知识笔记软件。它把富文本记录、树状结构、知识图谱、手写批注和 AI 辅助整理放在同一张可缩放画布上，帮助你从零散资料中搭建可理解、可扩展、可回顾的个人知识网络。

它适合学习复杂主题、拆解长文资料、梳理研究线索、整理项目知识，以及围绕 GraphRAG 做概念抽取和知识图谱实验。

## 核心功能

### 无限画布式笔记

在一张自由画布上放置文本块、树结构、图结构和手写线条。你可以拖动、缩放、选择、复制、粘贴、删除、置底，也可以一次移动多个画布对象，让资料不再被线性的文档结构限制。

### 富文本与图片记录

正文编辑基于富文本能力，支持标题、列表、代码块、图片粘贴、图片拖入和纯文本粘贴。图片会保存到本地应用目录，笔记内容以结构化数据持久化，方便后续备份和迁移。

### 树状结构与知识图谱

MindNote 同时支持两类结构化表达：

- 树状结构：适合整理层级、提纲、章节、分类和推导路径。
- 图结构：适合表达多对多关系、概念关联、因果关系和跨主题连接。

节点可以直接编辑富文本内容，边可以标注关系说明，并支持自动布局、手动拖拽、连线样式调整和图结构 JSON 查看。

### AI 辅助知识整理

MindNote 内置多种 AI 工作流，让笔记从记录工具升级为思考助手：

- 文本转图谱：将文本块内容抽取为 nodes / edges 图结构，并渲染到画布。
- 图谱总结：基于当前图结构生成核心主题、重要节点关系和后续展开方向。
- 节点学习：针对当前节点生成解释性注解，并自动扩展相关概念节点。
- 指定方向学习：用户可以给出研究方向，让 AI 按目标维度扩展知识网络。

### GraphRAG 图结构生成

MindNote 支持调用外部 GraphRAG 流程，从文本中生成 GraphML，再导入并拆分为可编辑的图结构。它适合把长资料、研究材料或项目文档快速转化为可探索的知识网络。

### 本地模型与云端模型

AI 能力支持两种模式：

- 本地模型：通过 llama-server 提供 OpenAI 兼容接口，可选择 CPU 或 GPU 运行时，并支持本地 embedding 服务。
- 云端模型：支持 OpenAI 兼容 API，可配置 DeepSeek、Qwen、GLM、OpenAI、Gemini 等服务，也可以填写自定义 Base URL、API Key 和模型名。

用户可以按隐私、成本和性能需求，在本地优先和云端能力之间切换。

### 多笔记本与多页面管理

MindNote 支持多个笔记本，每个笔记本包含多个页面。侧边栏可以新建、重命名、删除和切换页面；顶部支持笔记本选择、新建笔记本、导入 JSON 笔记本和全文搜索。

搜索会覆盖普通文本块、树节点和图节点，并能直接跳转到对应页面与内容位置。

### 本地保存与云同步

笔记本会自动保存为本地 JSON 文件，并采用临时文件与备份文件机制降低写入风险。登录后可通过 AWS Amplify 同步到云端；不登录时也可以使用本地免登录模式。

### 个性化外观与编辑体验

MindNote 支持浅色、深色和跟随系统主题，提供界面字体、正文字体、字号、正文颜色、画布颜色、图边颜色、边宽、连线样式和关系文字大小等设置。

画布支持滚轮滚动、缩放滑块、自定义滚动条、上下文工具栏、快捷撤销/重做和手写批注工具，让长期整理时的操作更顺手。

## 典型使用场景

- 学习新领域：把资料摘录、概念解释、层级结构和关联图放在同一张画布上。
- 阅读长文档：用 AI 抽取概念与关系，快速形成可编辑的知识图谱。
- 做研究选题：从一个节点出发，让 AI 按指定方向扩展相关概念。
- 整理项目资料：把需求、方案、风险、依赖关系和会议记录组织成可追踪结构。
- 构建个人知识库：用本地 JSON 保存长期资料，并按需同步到云端。

## 产品定位

MindNote 不是单纯的 Markdown 编辑器，也不是只画思维导图的工具。它更像一个面向知识构建的桌面工作台：先把信息放进画布，再用结构、关系和 AI 把信息变成可以继续思考的知识网络。

当前功能主要面向桌面端使用，尤其适合需要本地资料管理、本地 AI、图结构编辑和 GraphRAG 实验的用户。
